# CSS Styling in React

This project demonstrates different ways of applying CSS styling inside a React application.  
The main goal is to understand how styling works at the component level.

## Concepts Learned

- Applying inline styles in React components  
- Using external CSS files  
- Conditional styling based on state  
- Dynamically changing styles through user interaction  

## What This Project Showcases

This project showcases how React handles styling differently from traditional HTML.  
It highlights component-based styling and how styles can be updated dynamically when the state changes.

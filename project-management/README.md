# Project Management Application

This project is a simple project management interface where users can create and manage projects along with their related tasks.  
The main goal of this project is to understand component structuring, reusable inputs, and advanced React features.

## Concepts Learned

- Creating a project sidebar for navigation  
- Building reusable input components for forms  
- Managing project creation and updating the UI dynamically  
- Handling tasks related to each project  
- Using `useRef` to directly access DOM elements  
- Validating user input before submission  
- Displaying an error modal using `useImperativeHandle`  

## What This Project Showcases

This project showcases structured component design and better state organization.  
It demonstrates how to manage complex UI interactions, validate forms, and control child components imperatively.  
It also highlights how projects and their tasks can be handled efficiently within a React application.
